## Axios 사용법

---

### import

```javascript
import axios from "axios";
```

### POST

새로운 자원을 생성할 때 사용

```javascript
axios.post(url, data 객체)
```

### GET

자원을 요철할 때 사용

```javascript
axios.get(url);
```

### put

자원을 갱신할 때 사용

```javascript
axios.put(url, data 객체)
```

### DELETE

자원을 삭제할 때 사용

```javascript
axios.delete(url);
```
